# uteco_kip

**KIP**는 Knowledge Is Power의 의미를 가진, 퀴즈 앱이다.

------

KIP에서 주제, 문제수, 난이도 등을 선택하여 문제를 풀 수 있도록하고, 그에 대한 문제 해설까지 바로 제공한다.

#### 화면 구성
홈 화면, 스플래시 화면, 주제 선택 화면, 난이도 선택 화면, 문제 수 선택 화면, 문제 풀이 화면, 문제 해설 화면, 결과 화면 으로 구성된다.

기본적인 화면 제작 후 회원 기능, firebase 연동 등 세부 기능까지 추가해본다.
